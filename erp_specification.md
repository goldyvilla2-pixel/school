# OPENEDUCAT ERP - GLOBAL SPECIFICATION

## 1. DESIGN SYSTEM (BOLD MODERN AESTHETIC)
- **TYPOGRAPHY**: MONTSERRAT (GOOGLE FONTS).
- **CASING**: GLOBAL ALL CAPS ENFORCEMENT.
- **HIERARCHY**:
    - **H1 (HEADERS)**: BODY SIZE + 2 LEVELS (BOLD-800).
    - **H2 (SUBHEADERS)**: BODY SIZE + 1 LEVEL (BOLD-700).
    - **BODY**: BASE SIZE 14PX.
- **COLORS**: SIGNATURE ODOO PURPLE (#875A7B) AND DEEP CHARCOAL NAVY (#111827).

## 2. MODULE CONFIGURATION
- **CORE APPS**:
    - **STUDENTS**: CENTRAL MANAGEMENT OF STUDENT RECORDS.
    - **FACULTY**: ACADEMIC STAFF DIRECTORY.
    - **COURSES**: PROGRAM AND SUBJECT CATALOGUE.
    - **FEES** (RENAMED FROM INVOICING): FINANCIAL MANAGEMENT AND COLLECTIONS.
    - **REPORTS**: ANALYTICAL VISUALIZATIONS AND DOCUMENT DOWNLOADS (NEW).
    - **ADMISSIONS**: STUDENT ONBOARDING PIPELINE.
    - **ATTENDANCE**: DAILY TRACKING.
    - **TIMETABLE**: SCHEDULING.
- **DEPRECATED/REMOVED**: LIBRARY, CAMPUS (INFRASTRUCTURE).

## 3. INTERCONNECTIVITY PRINCIPLES
- **STUDENT ↔ FEES**: EVERY STUDENT RECORD LINKS DIRECTLY TO THEIR SPECIFIC FEE STATEMENT.
- **COURSES ↔ STUDENTS**: VIEW STUDENT LISTS DIRECTLY FROM COURSE MANAGEMENT.
- **ADMISSIONS ↔ STUDENTS**: ONE-CLICK CONVERSION FROM APPLICANT TO ENROLLED STUDENT.

## 5. RBAC & MULTI-BRANCH MATRIX
| ROLE | BRANCH ACCESS | FINANCIALS/REPORTS | EDIT (MODIFY/DELETE) | UPDATE (ADD NEW) |
| :--- | :--- | :--- | :--- | :--- |
| **SUPER ADMIN** | ALL | FULL ACCESS | ✅ YES | ✅ YES |
| **SUPER SUB** | ALL | ASSIGNED SCHOOL ONLY | ❌ NO | ❌ NO |
| **ADMIN** | ASSIGNED ONE | ASSIGNED ONLY | ❌ NO | ❌ NO |
| **ACCOUNTING** | ASSIGNED ONE | ASSIGNED + DOWNLOAD | ❌ NO | ✅ YES (FEES) |
| **COORDINATOR** | ASSIGNED ONE | ❌ NO ACCESS | ❌ NO | ✅ YES (CORE) |
